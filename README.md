# VL Notifier

> Unofficial browser extension for desktop notifications from VolumeLeaders.com

Get instant desktop notifications when new Trade Level Touches or Block Trades appear on [VolumeLeaders.com](https://www.volumeleaders.com).

Available for Firefox and Chrome.

## Features

- Real-time desktop notifications for new trades
- Sound alerts (optional)
- Ticker symbol, price, rank, and sector in notifications
- Trade type indicators (🔶 DP Sweep, 🟠 DP Trade, 🔷 Lit Sweep, 🔵 Lit Trade)

## Installation

### Firefox

1. Download the signed `.xpi` file from [Releases](../../releases/latest)
2. Open the file with Firefox to install

### Chrome

1. Download `vl-notifier-chrome.zip` from [Releases](../../releases/latest)
2. Extract to a folder
3. Go to `chrome://extensions/`
4. Enable "Developer mode" (top right)
5. Click "Load unpacked" and select the extracted folder

## Usage

1. Install the extension
2. Visit [Trade Level Touches](https://www.volumeleaders.com/TradeLevelTouches) or [Trades](https://www.volumeleaders.com/Trades)
3. Get notified automatically when new items appear

Click the extension icon to configure sound alerts and other settings.

## Requirements

- **Firefox:** 91.0 or later
- **Chrome:** 102 or later
- VolumeLeaders account
